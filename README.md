# Research Assistant Pro

التطبيق الاحترافي لإعداد بحوث المتابعة والدراسات العليا

## التخصصات
- التربية (Education)
- الخدمة الاجتماعية (Social Work)
- العلوم الإسلامية (Islamic Sciences)

## الميزات
1. **محرر RTL عربي احترافي**
   - دعم كامل للغة العربية
   - خطوط عربية محترفة
   - تنسيق بمعايير جامعية

2. **إدارة مراجع أكاديمية**
   - بحث في Crossref API
   - بحث في OpenAlex API
   - أساليب توثيق: APA, Chicago, Harvard

3. **عداد كلمات ديناميكي**
   - عداد لمجموع البحث
   - عداد لكل قسم
   - عرض التقدم النسبي

4. **إنشاء وتصدير مسندات**
   - PDF Export
   - DOCX Export
   - Google Docs Integration

## البنية التقنية

### Frontend
- React 18+
- TypeScript
- TailwindCSS
- Lexical (Rich Text Editor)
- RTL Support

### Backend
- Firebase Firestore
- Firebase Authentication
- Cloud Functions
- Google Docs API

### APIs
- Crossref API
- OpenAlex API
- Google Scholar (via scraping)

## أقسام البحث
1. Frontmatter (الغلاف الجامعي)
2. Abstract / الملخص
3. Introduction / مقدمة
4. Problem Statement / مشكلة البحث
5. Research Questions / أسئلة البحث
6. Objectives / أهداف
7. Previous Studies / الدراسات السابقة
8. Theoretical Framework / الإطار النظري
9. Methodology / منهجية البحث
10. Results / النتائج
11. Recommendations / التوصيات
12. References / المراجع

## التثبيت

```bash
git clone https://github.com/yourusername/research-assistant-pro
cd research-assistant-pro
npm install
npm run dev
```

## طريقة العمل
1. إنشاء مشروع بحثي
2. اختيار التخصص ونمط التوثيق
3. البدء بالكتابة والتنرسيق التلقائي
4. إضافة المراجع من قواعد بيانات أكاديمية
5. توليد طلب ببب أو DOCX
